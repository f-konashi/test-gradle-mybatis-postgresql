# springboot-gradle-sample
▼使用方法

	DB設定：
    	url: jdbc:postgresql://localhost/test
    	username: postgres
    	password: hmj
    	
	テーブル設定：
		・プロジェクトフォルダ直下のcreate_table.sqlで、テーブル作成・サンプルデータの登録が可能。
		・「drop_table.sql」で、登録した上記テーブルの削除が可能。

	ログイン方法：
		デフォルトで、下記ID・PWでログイン可能。
		ID: user
		PW: pass

	[Gradle]
	
	[Mybatis]
	
	[Mybatis　Generator]
	
	[postgresql]
